## 📁 프로젝트 상세 설명
APP 실행 후 영수증을 기반으로 데이터를 입력 후 N등분 계산을 진행합니다.

  <br>
  
  (1) 시작 화면  
  <img src="https://github.com/user-attachments/assets/84b4048a-a780-4d76-9e84-4d5092aa7176" width="20%">  
  -, + 버튼을 인원 수에 맞게 눌러 인원을 설정한 후 시작 버튼을 눌러 다음 화면으로 이동합니다.

  (2) 영수증 입력 화면  
  <img src="https://github.com/user-attachments/assets/60a9d635-e0f8-41d9-aa0f-e442604984a3" width="20%">  
  각 영수증에 상품 추가를 눌러 양식에 맞게 데이터를 입력합니다. 영수증에 항목을 채워넣으면 자동으로 총합이 계산되어 영수증의 좌측 상단에 표기됩니다. 좌측 상단의 글자를 눌러 영수증 이름을 설정할 수 있으며, 영수증 추가 버튼을 통해 영수증을 추가할 수 있습니다.

  (3) 정산 화면  
  <img src="https://github.com/user-attachments/assets/fbebc736-bfdd-48b2-85cc-7383f2cb9502" width="20%">  
  입력한 영수증들의 각 항목이 차례대로 가격과 함께 화면에 표기됩니다. 정산을 희망하는 인원들을 터치 후 적용 버튼을 눌러 해당 인원들에게만 N등분을 부여할 수 있습니다. 계산을 실수했을 경우 되돌리기 버튼을 통해 실수한 정산을 되돌릴 수 있습니다. OFF/ON 토글 버튼을 통해 각 인원의 버튼의 이름을 변경할 수 있습니다. 모든 정산이 끝나면 안내 메시지가 나오며 정산 완료 버튼을 통해 다음 화면으로 이동합니다.

  (4) 정산 화면  
  <img src="https://github.com/user-attachments/assets/0e63b22c-68fc-47fa-9cc0-5fbb933518b5" width="20%">  
  각 인원별 정산 금액을 확인할 수 있습니다. 각 인원 버튼을 눌러 상세 내역이 가능하며, 하단의 계좌 입력을 통해 계좌 번호 입력이 가능합니다. 저장 & 공유 버튼을 눌러 해당 화면을 캡처하여 갤러리에 저장할 수 있고, 다른 앱에 공유가 가능합니다.
  
<br>

## ✏️ 배운 내용
- Jetpack Compose를 사용해 선언형 UI 프로그래밍을 통해 APP 개발의 효율성을 극대화
- MVVM + Clean Architecutre 구조를 도입하여 APP 개발의 유지보수성 및 확장성을 확보
- Dialog를 Custom 후 사용하여 원하는 형태의 Dialog를 만들 수 있었고, 이를 통해 입력받은 데이터를 UI에 적용
- Hilt를 사용하여 ViewModel에 의존성 주입을 적용하였고, 생명주기 관린와 의존성 처리를 효율적으로 개선
- Navigation을 통한 화면 전환법 학습
